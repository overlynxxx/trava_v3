# TRAVA — Загородный клуб

Сайт загородного клуба «ТРАВА» с модулем бронирования.

## Что есть на сайте

- Hero с переключателем «🌿 Лето / ❄ Зима» (меняет фото в hero, бане, активностях и галерее)
- Полноширинный модуль бронирования: даты (календарь), гости, тип коттеджа, доп. услуги, расчёт стоимости
- Карточки коттеджей с каруселями фото (4 человека / на компанию)
- Секции: Главное, Проживание, Баня и джакузи, Активности (Лето / Зима / Круглый год), Корпоративы с формой КП, Зима (горнолыжные курорты рядом), Галерея, Отзывы, FAQ, Контакты с картой
- Переключатель RU / EN
- Tweaks-панель: стиль hero, палитра, типографика

## Структура

```
index.html       — корневой HTML
styles.css       — все стили
app.jsx          — главный компонент, монтаж в #root
sections.jsx     — все секции сайта
booking.jsx      — модуль бронирования
tweaks-panel.jsx — UI для tweaks
image-slot.js    — кастомный элемент <image-slot>
media/           — фотографии
```

## Запуск локально

Откройте `index.html` любым локальным веб-сервером (не file://, иначе модули не загрузятся):

```bash
# Python 3
python3 -m http.server 8000

# Node (если установлен http-server)
npx http-server -p 8000
```

Затем откройте http://localhost:8000.

## Деплой на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы (Add file → Upload files)
3. Settings → Pages → Source: Deploy from branch → Branch: `main`, `/` (root)
4. Через 1–2 минуты сайт будет доступен по `https://<your-username>.github.io/<repo-name>/`
